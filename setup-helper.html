Налаштування Trello Power-Up

Цей інструмент встановлює iframeConnectorUrl для твого Power-Up «Тайм-трекер» через Trello REST API. У новій версії Trello adminки це поле не редагується через UI.

Крок 1. Відкрий trello.com/app-key. Скопіюй Personal API Key (32 символи) і встав нижче.
Крок 2. На тій же сторінці натисни посилання "Token" (або "manually generate a Token") → погодься з доступом → скопіюй Token (довгий рядок) і встав нижче.
